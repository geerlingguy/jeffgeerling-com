Early in the morning, I finished upgrading all the Ansible playbooks used by the [Raspberry Pi Dramble](http://www.pidramble.com/) so my cluster of five Raspberry Pis would run faster and better on the latest version of official Raspberry Pi OS, Raspbian Jessie.
